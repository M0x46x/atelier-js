# Atelier JavaScript - TP

## Description
Ce dépôt contient les solutions des TPs JavaScript (TP1 à TP4).

## Structure
- TP1 : Structures de contrôle (if/else, switch, boucles)
- TP2 : Boucles avancées
- TP3 : Fonctions et expressions fléchées
- TP4 : Objets, tableaux, try/catch, JSON

## Exécution
1. Télécharger/cloner le projet.
2. Ouvrir les fichiers `.html` dans un navigateur (Chrome).
3. Pour voir les résultats : ouvrir la console (F12 → Console) lorsque l’exercice utilise `console.log()`.
4. Les exercices utilisant `prompt()` / `alert()` s’affichent automatiquement lors de l’exécution.
